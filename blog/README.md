# ブログ用 サービス紹介ページ（4種）

CMS（WordPress 等）の **固定ページ** にそのまま貼り付けて使える、自己完結型のHTML4種類です。

## ファイル一覧

| ファイル | 内容 | カラー |
|---|---|---|
| [`jinzai.html`](jinzai.html) | **人材サービス**（AI・IT人材マッチング、SES、採用支援） | 🔵 ブルー |
| [`education.html`](education.html) | **教育・スクール**（無限道場、DX/AI学習、実践型カリキュラム） | 🟠 オレンジ |
| [`community.html`](community.html) | **コミュニティ・イベント**（エイトカフェ拠点、交流・学び・企画） | 🟢 ティール |
| [`dev-support.html`](dev-support.html) | **開発・共創支援**（新規事業、DX、AI導入、共同プロジェクト） | 🟣 パープル |

## 特徴

- ✅ **自己完結**：1ファイルで完結。外部CSS・JS への依存なし
- ✅ **CMSにそのまま貼れる**：`<html>` `<head>` `<body>` タグなし、固定ページの本文に直接ペースト可能
- ✅ **テーマCSSと衝突しない**：全クラスに `eb-` プレフィックス、`.eb-page` でスコープ
- ✅ **画像不要**：ヒーロー画像は Unsplash の絶対URLを使用（差し替えも可能）
- ✅ **モバイル対応**：レスポンシブ（640px以下で自動的に縦並び）
- ✅ **CTA は 8ec.jp の問い合わせフォームへ自動リンク**（`https://8ec.jp/index.html#contact`）

## 各ページの構成（共通）

1. **ヒーロー** — タイトル＋キャッチコピー＋背景画像
2. **このサービスについて** — リード文
3. **提供サービス（4枚カード）** — 主要メニュー
4. **こんな企業におすすめ（4ケース）** — 利用シーン
5. **ご利用の流れ（4-5ステップ）** — オンボーディング
6. **CTA** — 無料相談・見積もり依頼

## 使い方

### WordPress 固定ページの場合

#### パターンA：ブロックエディタ（Gutenberg）
1. 固定ページを新規作成
2. 「+」→「カスタムHTML」ブロックを追加
3. ファイル全体をコピー＆ペースト
4. 「プレビュー」で確認 → 公開

#### パターンB：クラシックエディタ
1. 固定ページを新規作成
2. 「ビジュアル」→「テキスト」モードに切り替え
3. ファイル全体をコピー＆ペースト
4. 公開

### その他CMS（はてなブログ、note、Wix 等）の場合

「HTMLとして編集」「ソースを編集」「カスタムHTML」等のオプションから貼り付けてください。

## カスタマイズ

### ヒーロー画像の差し替え

各HTMLファイルの `<style>` 内、`.eb-hero` の `background:` 行を編集：

```css
.eb-hero{
  background:
    linear-gradient(135deg, rgba(11,43,122,.85) 0%, rgba(30,79,217,.7) 100%),
    url("https://your-image-url.jpg") center/cover;
}
```

### カラーテーマの変更

各ファイル先頭の `:root` 相当の `.eb-page` にあるCSS変数を変更：

```css
.eb-page{
  --eb-color:#1E4FD9;       /* メインカラー */
  --eb-color-dark:#0B2B7A;  /* ダーク（CTA背景等） */
  --eb-color-light:#5C82F0; /* ライト */
  --eb-color-tint:#E8EFFF;  /* 薄い背景・チップ用 */
  ...
}
```

### CTA リンク先の変更

各ファイル末尾の CTA セクションの `<a class="eb-btn" href="...">` の `href` を変更してください。

```html
<a class="eb-btn" href="https://8ec.jp/index.html#contact">無料相談・見積もり</a>
<a class="eb-btn eb-ghost" href="tel:0364335025">03-6433-5025</a>
```

## 注意事項

- **WordPress自動pタグ挿入**：一部のテーマでは `wpautop` で `<p>` が自動挿入され見た目が崩れる場合があります。`Disable wpautop` プラグイン、もしくは「カスタムHTML」ブロックを使うと回避できます。
- **既存テーマCSSとの衝突**：`.eb-page` 配下にスコープしていますが、テーマが極めて強いCSSを使っている場合は崩れる可能性があります。その場合は各CSSルールに `!important` を追加してください。
- **画像のローカル化**：本番サイトの読み込み高速化のため、Unsplashから画像をダウンロードしてWordPressメディアにアップロードし、URLを差し替えることを推奨します。
